## Overview
Advanced dashboard for World of Warcraft guilds to analyze raid performance.

- OAuth2 authentication (Battle.net / Google)
- Monte Carlo DPS simulation powered by Go
- Raider.IO & WarcraftLogs API aggregation
- Real-time data visualization
- Redis caching layer for performance

👉 Goal: optimize raid composition and performance.
